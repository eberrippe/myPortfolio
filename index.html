Other

Alongside the work I do for clients, I also have an interest in the devlopment of new interactive media approaches.
Examples include:

Neural network for a realtime magnet based 3D tracking system

Integrated with TensorFlow, sent through OSSC, visualised in Unity.


role: NN-architecture, development

DESC: A Vive-Tracker glued on top of a magnet to generate positions as training values for the measured magnet data.

Outcome: The offset between the predicted magnet positons and the actual Vive-tracked positons is visualized live in Unity in a XYZ-perspective.

Neural network to identify food types

Integrated with TensorFlow - it knows when you're eating French fries!


role: NN-architecture, development

Feed an image to the software and hope it's right!

The software was trained to identify 'chocolate_cake, edamame, french_fries, greek_salad, red_velvet_cake'.

Facescan character editor in Unity

A FaceGen (Singular Inversions 2021) based character editor in a game engine.


role: workflow, hair, development

The project was used to enhances the models created from FaceGen in a realtime engine. Optimises skin, eyes and hair.
